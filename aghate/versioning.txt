le version 2.0 est stable pour l'instant

Revision

Numero de revision corresponds le date de dernière modifications de la base de donnèes ou scripts
format du date est AAAAMMJJ

Le variable $version_grr_RC de la script commun/include/misc.inc.php doit etre modifé a chaque revision

si cette revision ne coressponds pas dans la base agt_config variable version_RC, 
le script admin_maj.php doit etre executé par administrateur pour mettre a jour les dernière modification.
